YouTube Video Finder & Sharer

Quickly find and share your videos

Setup Instructions

  1. Get YouTube API Credentials:
    1. Go to Google Cloud Console
    2. Create a project or select existing one
    3. Enable YouTube Data API v3
    4. Create OAuth 2.0 credentials
  2. Add Environment Variables:
    YOUTUBE_CLIENT_ID=your_client_id
    YOUTUBE_CLIENT_SECRET=your_client_secret
    REDIRECT_URL=your_deployed_url/auth/callback
  3. Deploy to Hosting Platform:
    • Vercel (Easy deployment)
    • Heroku
    • Render
    • Or your own server with Node.js
  4. After deployment, visit your site and connect your YouTube account!

This is a template repository. The full working version requires YouTube API integration.

For the complete version with authentication, see the repository README.
